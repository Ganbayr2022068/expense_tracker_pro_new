✨ Features
</div>
<div align="center">

| Icon | Feature | Description |
|------|---------|-------------|
| 💸 | **Add & Categorize Expenses** | Log transactions with amount, category, date, and notes |
| 📊 | **Dashboard Overview** | See total spending at a glance with summary cards |
| 📈 | **Visual Charts** | Pie and bar charts to visualize spending by category |<img 
| 🕓 | **Transaction History** | Browse, search, and filter all past expenses |

🛠️ Getting Started
  </div>
  <div align="center">
Prerequisites
</div>
<div align="center">
Flutter SDK >=3.0.0 
</div>
<div align="center">
Dart SDK >=3.0.0
</div>
<div align="center">
Android Studio / VS Code with Flutter plugin
</div>
<div align="center">
A connected device or emulator
</div>



<div align="center">
Installation
  </div>
  
<div align="center">
1. Clone the repository
git clone https://github.com/Ganbayr2022068/expense_tracker.
  </div>
<div align="center">
2. Navigate into the project
cd expense_tracker
  </div>
<div align="center">
3. Install dependencies
flutter pub get
  </div>
<div align="center">
4. Run the app
flutter run
</div>
